# Expense Tracker

A simple, single-page expense tracking application built with vanilla HTML, CSS, and JavaScript. Data is persisted in the browser using localStorage.

## Features

- Add, edit, and delete expenses with name, amount, category, and date
- Filter expenses by category
- Search expenses by name
- Dark mode toggle with persistent preference
- Export expenses to CSV
- Responsive design for mobile and desktop

## Usage

Open `index.html` in any modern web browser. No build tools or server required.
